# QLineSec
# Amaç
Bir tarayıcı eklentisini güvenlik açısından analiz etmek ve potansiyel güvenlik açıklarını raporlamak.

## İnceleme Kapsamı
- Manifest izinleri
- Content script analizi
- Background script güvenliği
- Mesajlaşma mekanizması (runtime.onMessage)
- Harici ağ istekleri
- Zararlı kod kalıpları (eval, innerHTML vb.)
